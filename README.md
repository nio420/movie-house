# Movie House

A modern movie web application built with **React** and **Vite**, designed for movie lovers to browse, view, and download movies seamlessly. Fully responsive with smooth animations and optimized performance.

---

## Features

- Browse and view movie collections
- Detailed movie pages with download functionality
- Fully responsive design for desktop and mobile
- Smooth animations with Tailwind CSS
- Optimized rendering using `React.memo`

---

## Tech Stack

- **Frontend:** React, Vite
- **Styling:** Tailwind CSS
- **Routing:** React Router
- **Icons:** Lucide React
